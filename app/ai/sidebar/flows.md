# 사이드바 플로우(초안)

## 공통
- 입력: 현재 문서 컨텍스트(선택/헤더/메타) 스냅샷
- 출력: 미리보기 패널(적용 전 확인), Evidence 앵커 표시
- 로컬‑퍼스트: Ollama 우선, 실패/미설치 시 동의 후 원격(토글)

## 요약(Summarize)
1) 컨텍스트 수집 → 2) 하이브리드 검색 TopK → 3) 템플릿 생성 → 4) 미리보기 → 5) 적용

## Q&A
1) 질의 파싱(WRITE/STRUCTURE/… 라우팅 후보) → 2) 검색 → 3) 생성 → 4) 앵커 렌더 → 5) 적용/복사

## 편집(EditStyle)
1) 선택 범위 확보 → 2) 스타일 지시어 적용 → 3) 미리보기 diff → 4) 적용(Undo 경계)
