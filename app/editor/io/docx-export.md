# Docx 내보내기 범위/매핑(초안)

> 기능 토글: docxExport (기본 OFF, M2)

## 범위
- 제목/헤더/본문/리스트/표/코드블록/링크
- 주석/각주/머릿글/바닥글: 제외(M2+)

## 매핑 규칙
- Heading H1–H3 → 스타일 Heading1–3
- 리스트(불릿/번호) → 대응 목록 스타일, 중첩 2레벨 보장
- 표 → 기본 표(헤더 행 포함), 병합/정렬 제외
- 코드블록 → 고정폭 스타일
- 링크 → 하이퍼링크(표시 텍스트 유지, 내부 IRI는 표시 텍스트만)

## 메타
- 문서 속성: title, createdAt, updatedAt
- 태그/속성: 문서 말미 부록 섹션(옵션)
