# 단축키 맵(초안)

> 목표: 마우스 없이 제목/리스트/테이블 입력, 명령 팔레트와 언급, Tab‑fill 수락 제스처 제공.

## 글로벌
- Ctrl+P: 최근 문서 전환
- Ctrl+/: 명령 팔레트(`/`)
- Ctrl+F: 문서 내 검색

## 서식/블록 삽입
- Ctrl+Alt+1: H1
- Ctrl+Alt+2: H2
- Ctrl+Alt+3: H3
- Ctrl+Shift+L: 불릿 리스트 토글
- Ctrl+Shift+O: 번호 리스트 토글
- Ctrl+Shift+T: 테이블 삽입(기본 3×3)
- Ctrl+` : 코드 블록 토글

## 명령/언급
- `/`: 명령 팔레트 열기(포커스 내)
- `@`: 문서/개체 언급 자동완성
- `[[`: 링크 검색 드롭다운(신규 문서 만들기 포함)

## Tab 기반 Fill(FIM)
- Tab: 제안 수락(기본)
- →: 제안 수락(보조)
- 입력 타이핑: 충돌 시 제안 취소 규칙 적용
- Esc: 제안 숨기기

## 테이블 내 네비게이션
- Tab: 다음 셀, Shift+Tab: 이전 셀
- Enter: 다음 행, Shift+Enter: 이전 행

## 접근성
- Alt+[ / Alt+]: 사이드바 폭 조절
- F6: 영역 순환(에디터↔사이드바↔상태바)

## 참고(수용 기준 매핑)
- 팔레트 키로 `/` 명령 호출 가능해야 함
- H1/리스트/테이블 삽입을 키보드로 수행 가능해야 함
- Tab/→로 고스트 제안 수락 가능해야 함
