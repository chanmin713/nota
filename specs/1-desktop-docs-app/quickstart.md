# 퀵스타트(초안)

## 1) 문서 작성/편집
- 새 문서 생성 → Ctrl+Alt+1 로 제목(H1)
- 불릿 리스트: Ctrl+Shift+L, 테이블: Ctrl+Shift+T(3×3)
- 마크다운 자동 인식: '#', '-', '1.' 즉시 구조화
- Tab 으로 고스트 제안 수락(문장 단위 기본)

## 2) AI 사이드바(오프라인)
- 사이드바 열기 → Summarize 실행
- 네트워크 차단 상태에서도 응답 확인(10초 내 목표)
- 근거 앵커를 확인하고 적용 전 미리보기

## 3) / 명령 & @ 언급
- '/' 로 명령 팔레트 → '요약', '아웃라인' 등 실행
- '@' 로 문서/개체 언급, '[[‘ 로 링크 삽입(새 문서 만들기 지원)

## 4) 태그/속성/검색
- 태그/속성 추가 → 검색 패널에서 필터 적용
- type 불일치 값 입력 시 즉시 경고(텍스트/숫자/날짜)

## 5) 버전 & 보안(요약)
- 문서 저장 시 버전 이력 기록, 차이 보기/롤백
- `.nota` 는 기본 암호화(AES‑256‑GCM), 키는 OS 보안 저장소 연동

> 성능/수용 시나리오는 QA 문서에서 측정 지표로 검증됩니다.

## 오프라인 요약 타이밍 스크립트(≤10s)
- 네트워크 오프 → 현재 문서 열기 → 사이드바 Summarize → 완료 시간 기록
- 3회 반복 평균/95p 산출
